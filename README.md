# gimped-REST-API
This is a template ive been working on for a while. its not really a REST API because its made to be useable with only a browser.

# Dependencies
    NodeJS
        - bcrypt
        - body-parser
        - express
        - knex
        - sqlite3

# Installation
```
git clone https://github.com/Nodnarb12500/gimped-REST-API
cd gimped-REST-API
npm i
npm start
```
